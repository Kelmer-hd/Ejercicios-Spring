## Ejercicio: Gestión de Tareas

### Requisitos Funcionales:

1. Crear una aplicación web utilizando Spring Boot que permita gestionar una lista de tareas.
2. La aplicación debe tener las siguientes funcionalidades básicas:
    - Crear una nueva tarea
    - Listar todas las tareas existentes
    - Marcar una tarea como completada
    - Eliminar una tarea

### Especificaciones Técnicas:

1. Utilizar Spring Boot como framework principal.
2. Crear una entidad "Tarea" con los siguientes atributos:
    - id (identificador único)
    - título
    - descripción
    - estado (pendiente/completada)
    - fecha de creación
3. Implementar un servicio y un repositorio para manejar las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de las tareas.
4. Crear controladores REST para exponer los endpoints de la aplicación.
5. Utilizar Postman o una herramienta similar para probar los endpoints de la aplicación.

### Entregables:

1. Código fuente de la aplicación Spring Boot en formato Markdown.
2. Instrucciones detalladas sobre cómo ejecutar y probar la aplicación.
